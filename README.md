# Alumni and Event Management Portal

## Project Overview

The **Alumni and Institutional Event Management Portal** is a web-based platform designed to help educational institutions manage alumni records, departmental activities, and institutional events in an organized and secure manner.

The system provides **role-based access** for administrators and department authorities to maintain alumni information, track placement details, manage institutional events, and generate reports.

This platform improves communication between alumni and institutions while enabling efficient data management through a centralized digital system.

* Vite

### Backend / Database

* Supabase (PostgreSQL Cloud Database)
* REST API Integration

### Development Tools

* Node.js
* npm Package Manager

## System Features

* Alumni directory management
* CSV import/export for alumni records
* Department-wise alumni filtering
* Placement status tracking
* Event management module
* Role-based authentication
* Dashboard and reporting features

## Installation and Setup

### 1. Clone the repository

```
git clone https://github.com/Jenson1124/christ-alumni-portal.git
```

### 2. Navigate to the project folder

```
cd christ-alumni-portal
```

### 3. Install dependencies

```
npm install
```

### 4. Start the development server

```
npm run dev
```

### 5. Open in browser

```
http://localhost:5173
```
